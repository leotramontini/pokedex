## Get Start :running:

Clone the project, in the project directory, you can run:

```
$ npm install
```
or 

```
$ yarn install
```

to install dependecies after run:

```
$ npm start
```

or

```
$ yarn start
```

to start application, open http://localhost:3000 to view it in the browser.
